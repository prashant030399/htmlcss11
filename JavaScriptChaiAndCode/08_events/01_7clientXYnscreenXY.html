Move your mouse to see its position.

e.clientX n e.clientY Gives the x and y coordinates of the mouse pointer relative to the viewport (visible part of the webpage). e.screenX n e.screenY Provides the x and y coordinates relative to the entire screen (including browser toolbars, OS elements, etc.). e.pageX n e.pageY Gives the x and y coordinates relative to the entire document (including scrolling). e.offsetX n e.offsetY Provides x and y coordinates relative to the target element where the event occurred. client -> Used in handling mouse events like mousemove, click, etc. screen -> Used when dealing with multi-screen setups or tracking the exact screen position of an event. page -> Used when dealing with scrolling pages, like positioning elements dynamically. offset -> Useful for canvas drawing, drag-and-drop inside an element, and interactive elements.
